# Age Calculator

A simple Age Calculator built using HTML, CSS, and JavaScript.

## Features

* Calculate age in years
* Calculate age in months
* Calculate age in days
* User-friendly interface
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript
